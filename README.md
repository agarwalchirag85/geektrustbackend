                                                                                                                        
This is a geektrust backend project(Tame Of THrones) designed using node.js.
For input I have used a file system.

After unzipping the file run the below command to install the necessary packages.
npm install --silent

Use this to run the code after npm install.
npm start --silent <absolute_path_to_input_file>

Use this command to run the testcases.
npm test
